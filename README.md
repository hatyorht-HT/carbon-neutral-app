# 编程练习平台

为电子信息工程学生打造的 C 语言和 Python 刷题平台，提供高质量的编程实战题目和详细的错误分析。

## 功能特性

- **题库首页**：浏览所有题目，按难度、标签筛选
- **个人错题本**：记录错误类型，智能推荐重练
- **个人中心**：查看学习数据，能力图谱
- **题目详情**：详细的题目描述，代码编辑器

## 技术栈

- HTML5
- Tailwind CSS v3
- Font Awesome
- JavaScript

## 如何使用

1. 直接在浏览器中打开 `index.html` 文件
2. 浏览题目列表，点击"开始练习"进入题目详情页
3. 在代码编辑器中编写代码，点击"运行"测试
4. 查看个人错题本，分析错误原因
5. 在个人中心查看学习数据和进度

## 页面结构

- `index.html` - 题库首页
- `notebook.html` - 个人错题本
- `profile.html` - 个人中心
- `problem.html` - 题目详情页

## 特色功能

1. **智能重练推荐**：根据遗忘曲线推荐重练题目
2. **错误分类分析**：详细记录语法错误、逻辑错误等
3. **学习数据可视化**：提交热图、能力图谱
4. **代码编辑器**：支持多种编程语言
5. **实时反馈**：运行结果实时显示

## 注意事项

- 本平台为前端演示版本，不包含后端功能
- 所有数据均为模拟数据
- 代码编辑器仅为展示，不支持实际运行代码
